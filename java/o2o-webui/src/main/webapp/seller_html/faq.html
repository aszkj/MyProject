• 1.如何上架、下架商品？

    1).在合伙人 APP首页后，点击“商品管理”；
    2).进入商品管理列表页后，选择商品，点击“下架”，即可将商品下架；
    3).同理在已下架商品页面，点“上架”即可将商品上架。

  • 2.如何搜索商品？

    1).在合伙人 APP首页后，点击“商品管理”；
    2).进入商品管理列表页后，点击“搜索”，进入搜索页面；
    3).在搜索页面中的输入框中，输入“商品名称”或“商品条形码”，点击“搜索”按钮，即可完成商品的搜索。

  • 3.如何进行快速接单？

    1).打开一里递合伙人 APP；
    2).进入合伙人 APP首页后，点击“待发货订单”或者“订单管理”；
    3).进入订单管理列表页后，点击需要处理的订单； 4).进入订单详情页后，点击“确认接单”，即可完成接单。

  • 4.使用“收货码”进行收货？

    1).打开一里递合伙人 APP；
    2).进入合伙人 APP首页后，点击 “订单管理”，进入订单列表页；
    3).进入订单管理列表页后，点击“收货码”按钮；
    4).在弹出的输入框中，输入“收货码”，点击“确定”，进入该收货码对应的订单详情页，即可成功收货。

  • 5.订单状态的说明？

    1).“待接单”是指：付款成功等待合伙人处理的订单；
    2).“未发货”是指：合伙人接单但未配送的订单；
    3).“配送中”是指，合伙人正在配送的订单；
    4).“已完成”是指：交易成功的订单；
    5).“已取消”是指：交易取消的订单。

  • 6.买家付款后，门店可以不接单吗？

    1).打开一里递合伙人 APP；
    2).进入合伙人 APP首页后，点击“待发货订单”或者“订单管理”；
    3).进入订单管理列表页后，点击需要处理的订单；
    4).进入订单详情页后，点击“确认接单”，可选择不接单。

  • 7.合伙人如何联系买家？

    1).打开一里递合伙人 APP；
    2).进入合伙人 APP首页后，点击 “订单管理”，进入订单列表页；
    3).进入订单管理列表页后，点击“电话标志”按钮，即可联系该订单的买家

  • 8.如何进行商品及商品库存的调拨？

    1).打开一里递合伙人 APP；
    2).进入合伙人 APP首页后，点击“我要调货”；
    3).进入调货商品管理页后，点击“+”按钮将商品加入调拨单；
    4).再点击“加入调货单”，进入购物车页面，点击“生成调拨单”，即完成商品的调拨。

  • 9.免责声明

    对于恶意刷单、恶意索赔等违反诚实信用原则的行为，一里递有权拒绝相关的补贴及无理赔偿，甚至取消合伙人入驻资格，并将相关证据提交给工商部门、公安部门依法处理。
