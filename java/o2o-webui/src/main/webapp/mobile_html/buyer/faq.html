常见问题

  • 1、一里递的营业时间？

    A：一里递客服的服务时间是9:00-21:00，合伙人的配送时间以每个门店的服务时间为准。

  • 2、可配送哪些城市？

    A：一里递已经在深圳龙华新区开通配送服务，其他城市及地区将会陆续开通，敬请关注！

  • 3、我如何下单购买商品？

    A：1）打开一里递客户端；
    2）进入一里递页面后，定位/选择地址，选购商品加入购物车；
    3）进入购物车确认商品信息，完善地址和收货人信息，点击“去结算”；
    4）进入订单结算页面，选择支付方式、优惠劵等，确认支付金额后，点击“确认付款”。

  • 4、如何进行支付？

    A：目前我们只支持在线支付，在线支付包括：支付宝支付和微信支付。

  • 5、下单后发现信息填写错误怎么办？

    A：如果合伙人尚未接单，您可以取消订单后重新下单；如果合伙人已接单，您可以拨打客服电话4001-333-866帮您取消订单，然后重新下单。

  • 6、收货码怎么使用？

    A：您收到商品清点无误后，请将收货码提供给合伙人；若您还未收到商品，请不要提供收货码。

  • 7、收货码短信不小心删除了怎么办？

    A：收货码会在您的订单中显示，请您登陆“我的”账户，点击“我的订单”，进入订单详情查看。

  • 8、提示地址不在配送范围？

    A：出现这样的提示，是因为您所在的地址超出合伙人的配送范围，无法为您提供配送服务；为了保证您良好的服务体验，我们建议您选择收货地址附近的合伙人进行配送服务。

  • 9、退款说明 - 取消订单/退货后，钱怎么返还？

    A：在线支付的订单，款项一般会返还到原支付账户。

  • 10、退款说明 - 退款多久可以到账？

    A：一般是3-5个工作日到账；此退款周期仅供参考，具体退款周期会受银行、支付机构等因素影响。

  • 11、退款说明 - 为什么退的金额跟我的订单金额不一致？

    A：退款的金额是退回该订单的实际支付金额：
    1）取消订单：会退回该订单的实际支付金额；该订单使用优惠劵会退回，在有效期内可以继续使用；
    2）售后退款/退货：会退回该商品的实际支付金额，优惠劵不会退回。

  • 12、免责声明？

    A：对于故意套单、恶意索赔等违反诚实信用原则的行为，一里递有权取消订单、作废优惠劵、取消活动参与资格及拒绝相关的无理赔偿。
